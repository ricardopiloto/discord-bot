# Local de Ferimento (Hit Location)

Depois de acertar, determine **onde** o golpe landed para armadura e tabela de críticos.

## Método rápido (regra padrão)

**Inverta os dígitos** do resultado do teste de acerto no d100 e consulte a tabela:

| d100 invertido | Local |
|---|---|
| 01–09 | Cabeça (Head) |
| 10–24 | Braço secundário / esquerdo |
| 25–44 | Braço primário / direito |
| 45–79 | Corpo (Body) |
| 80–89 | Perna esquerda |
| 90–00 | Perna direita |

### Exemplo

Rolou **23** para acertar → invertido = **32** → Braço primário.

## Armadura por local

Cada peça de armadura protege locais específicos. Use os **PA** daquele local ao reduzir dano.

## Críticos por local

A tabela de Ferimentos Críticos é consultada **por local atingido**. O mesmo número de crítico pode ter efeitos diferentes na cabeça e na perna.

## Chamada de tiro (opcional)

Atacar um local específico impõe penalidade de dificuldade (ex.: −10 cabeça, −20 olho). O Mestre pode exigir chamada de tiro para efeitos narrativos precisos.

Ver também: **Dano e Ferimentos**, **Ataque Corpo a Corpo**, **Ataque à Distância**.
