# Bertoldt Lamprecht

Acadêmico. Óculos redondos, bem vestido, completamente deslocado numa floresta perigosa. Olhos rápidos demais — cataloga tudo mentalmente como se estivesse escrevendo uma dissertação. Fala pouco; quando fala, é preciso. Acredita que conhecimento é autoridade.
