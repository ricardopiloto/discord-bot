# Joost

Ladrão. *O Furtivo*. Chegou ao domínio de [[Johans (Lorenz)|Johans]] sem ser visto — nenhum passo, nenhum anúncio, apenas estava lá. Manto azul-escuro, rosto oculto pelo capuz. Magro demais para parecer perigoso; perigoso demais para ser ignorado. Johans brincou: *"Recomendo vigiar os bolsos quando você está por perto."*
